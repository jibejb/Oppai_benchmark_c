# Oppai_benchmark_c

Oppai_benchmark rewrite in C.

## Usage:

Dependences: ISO C99 OpenGL(3.3 or later) glfw3(libglfw-dev) ncurses(libncurses-dev) libpthread

```bash
git clone https://github.com/jibejb/Oppai_benchmark_c.git
cd Oppai_benchmark_c
mkdir build
cd build
cmake ..
make
./oppai_benchmark -h
```
